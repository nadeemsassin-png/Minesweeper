# Minesweeper

A console-based implementation of the classic Minesweeper game written in C++.

## Features

* First-click safe mine generation
* Automatic blank-cell expansion
* Flag placement system
* Win and loss detection
* Persistent high scores
* Easy, Medium, and Hard difficulties

## Technologies Used

* C++
* STL Containers
* File I/O
* Random Number Generation
* Object-Oriented Programming

## Concepts Demonstrated

* Data structures
* Recursive/graph-style exploration
* Input validation
* State management
* Persistent storage
* Console user interfaces

## How to Build

g++ main.cpp -o minesweeper

## How to Run

./minesweeper
